El Laboratorio Contiene 2 ejecutables y un archivo.

***********************************************************
                          Reciver
***********************************************************
Es el Servidor, es quien recibe los archivos de Sender, por
esto es el que debe ejecutarse primero con el comando:

Ejecutar:
  python reciver.py

***********************************************************
                         Sender
***********************************************************
Es el Cliente, es quien envia los archivos a Reciver, por
esto se le debe entregar la ruta y nombre de un archivo a
enviar, por ejemplo con el archivo de prueba seria
el siguiente procedimiento:

Ejecutar:
  python sender.py

En la aplicacion aqui nos pide el nombre o la ruta de un 
archivo escribimos file.jpg
En el Programa:
  nombre y ruta del archivo a subir-> file.jpg


Los archivos enviados pueden tener un tamaño arbitrario.
